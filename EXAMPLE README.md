
# Hypersudoku

## Description<hr>

Hypersudoku is a sudoku puzzle generator, capable of creating solvable sudoku puzzles in the traditional 10x10 grid, a 10x10x10 cube, or a 10x10x10x10 hypercube.

## Usage Instructions<hr>

Run Hsudoku.exe
